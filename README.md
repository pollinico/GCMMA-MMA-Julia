# GCMMA-MMA-Julia

The Julia code provided in this respository implements the Method of Moving Asymptotes (Svanberg, 1987), which is based on the GCMMA-MMA-code originally written by Krister Svanberg for MATLAB. 
The original code can be obtained from http://www.smoptit.se/ and is distributed under the GNU General Public License.   

If you choose to download and use this code, Krister Svanberg kindly requests that you send him an email, mentioning your identity and explaining your intended usage of the code. You can find his email address on his website.
